# Movie Manager

A full-stack application for managing your movie collection.

## Features

- User authentication
- Movie collection management
- Integration with TMDB API
- New releases browsing
- Movie search functionality
- Responsive design

## Tech Stack

- Frontend:
  - React
  - Material-UI
  - Apollo Client
  - React Router

- Backend:
  - Node.js
  - Express
  - GraphQL
  - MongoDB
  - JWT Authentication
